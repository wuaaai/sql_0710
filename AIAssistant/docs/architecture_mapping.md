# 架构映射说明

## 一、总览

该骨架按照思维导图中的中心主题"智能助手系统设计思维导图"拆分为 8 个一级模块，并使用 `src/` 作为主包入口。

## 二、一级模块与导图节点对应关系

### 1. 用户请求生命周期
- 用户问题输入 → 请求预处理 → 双执行线路 → 结果融合与分析 → LLM 总结回答 → 会话记忆存储 → 结果反馈与优化

### 2. RAG 知识库链路
- 知识源构建 → 智能分块 → Embedding 与存储 → 知识图谱 → 检索增强 → 输出知识片段

### 3. Text2SQL 查询链路
- 槽位解析 → 插件系统 → 槽位补全 → 槽位确认 → 智能选表 → SQL 生成与优化 → SQL 执行 → 异常处理与重试 → 返回执行结果

### 4. 结果融合与分析
- 结果对齐 → 优先级判断 → 冲突检测 → 可信度评分 → 引用溯源 → 综合分析

### 5. 上下文记忆
- 会话记忆 → 历史会话 → 用户画像 → 最近问题与最近回答 → 最近 SQL → 知识记忆 → 长期记忆 → 记忆管理

### 6. 异常处理与优化策略
- 召回不足 / 槽位缺失 / SQL 执行失败 / 知识冲突 / 上下文处理异常 / 系统异常

### 7. 数据处理中心
- 数据源管理 → 数据同步 → 数据清洗 → 数据质量评估 → 元数据管理 → 表结构理解 → 向量数据维护 → 定时维护任务 → 智能读表支持 → 数据监控

### 8. 后台管理系统
- 知识库管理 / 插件管理 / 槽位规则管理 / Prompt 管理 / SQL 模板管理 / SQL 规则管理 / 用户与权限 / 日志与审计 / 数据看板 / 系统配置

## 三、推荐调用关系

`interfaces` -> `use_cases` -> `orchestrators` -> `services` -> `domain/contracts` -> `repositories/infrastructure`
